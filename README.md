# Spotify Clone (Frontend)

A responsive Spotify-inspired music streaming UI built using HTML, CSS, and JavaScript. This project replicates the look and feel of Spotify's interface, focusing on modern design, layout structuring, and interactive frontend features.

---

## Project Overview

This project is a frontend clone of Spotify designed to practice UI/UX design and frontend development skills. It includes a music player interface, playlists, and interactive controls to simulate a real music streaming experience.

---

## Features

* Responsive User Interface
  Clean and modern layout similar to Spotify

* Music Player UI
  Play, pause, and navigation controls

* Playlist Section
  Display songs and albums

* Navigation Sidebar
  Home, Search, and Library sections

* Interactive Elements
  Hover effects and dynamic UI behavior

---

## Technologies Used

* HTML5
* CSS
* JavaScript

---

## Project Structure

* index.html – Main structure of the website
* style.css – Styling and layout design
* script.js – Interactivity and player controls
* assets/ – Images, icons, and audio files
